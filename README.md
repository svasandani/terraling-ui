A React front-end for Terraling, based on a dynamic activity feed instead of a static homepage.

## Installation
- Clone the repo

  `$ git clone github.com/svasandani/terraling-ui`

- Install dependencies

  `$ npm install`

- Start the server

  `$ npm start`
